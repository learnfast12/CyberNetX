# CyberNetX
**Detect. Trace. Neutralize.**

One platform, two shields — stopping banking fraud before it reaches the customer, and tracing the money if it doesn't.

## Modules

- **ShieldNetX** — Customer Shield. Real-time phishing/smishing detection (browser extension + Android app with Accessibility Service interception).
- **TraceNetX** — Bank Shield. AML mule-account detection and money-trail tracing using graph intelligence + ML ensemble.

Built by Team OMEGA 404 — Sri Sairam Engineering College, Chennai (OWASP Sri Sairam) — for IDBI Innovate 2026.
